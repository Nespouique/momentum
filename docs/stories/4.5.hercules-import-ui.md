# Story 4.5: Interface d'import Hercules (Optionnel)

## Status: To Do (À implémenter plus tard si nécessaire)

## User Story

**As a** user,
**I want** a UI to import my Hercules data,
**So that** I can migrate without using command-line scripts.

## Status

**DEFERRED** - À implémenter plus tard si nécessaire.

Pour le moment, l'import se fait via scripts CLI (Stories 4.1, 4.2, 4.3).

## Acceptance Criteria (Future)

1. Page `/settings/import` accessible depuis les paramètres
2. Upload du fichier `.db` Hercules
3. Prévisualisation des données avant import :
   - Nombre d'exercices à créer
   - Nombre de séances à importer
   - Nombre de mensurations
4. Mapping visuel des exercices (drag & drop ou dropdown)
5. Bouton "Importer" avec progress bar
6. Rapport d'import final
